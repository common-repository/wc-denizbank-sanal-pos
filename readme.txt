== Description ==
- Garsoft entegrasyon ekibi tarafından geliştirilen bir açık kaynak WooCommerce modülüdür.

- Bu eklenti ile ödemelerinizi 3d ile alabilirsiniz.

- Modülümüz woocommerce ile %100 uyumludur.

Requirements
PHP 5.4 and greater.
WooCommerce 3.5 requires WordPress 3.5+

== Installation ==
- Denizbank sanal pos eklentisini yükleyebilmek için indirdiğiniz dosyaları herhangi bir FTP programı ile public_html/wp-content/plugins klasörüne yükleyin.
- wp-admin eklentiler bölümünden Payment Gateway for Denizbank eklentisini aktif edin.
- wp-admin > Woocommerce > Ayarlar > Ödemeler bölümünden Denizbank Sanal Pos etkin hale getirin.
- siteadi.com/wp-admin/admin.php?page=wc-settings&tab=checkout&section=denizpos sayfasından banka tarafından size verilen entegrasyon bilgilerini girin. (mağaza numarası, kullanıcı kodu, şifre, 3d anahtarı)
